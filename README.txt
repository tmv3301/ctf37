이 문제는 네 번의 실행을 거쳐 각각 하나씩 총 4개의 HEX 값을 획득해야 한다.
이 4개의 HEX 값을 특정 규칙에 따라 조합하여 최종 플래그를 알아내야 한다.
조합 규칙은 코드 분석을 통해 유추할 수 있는 규칙이다.

./problem 1, ./problem 2, ./problem 3, ./problem 4 이러한 방식으로 네 번의 실행을 진행해야 한다.
각각의 실행에 대해 Key를 입력해야 하고, 정답 Key를 입력하면 8자리의 HEX 값이 출력된다.
잘못된 Key가 입력될 경우 "denied" 문구가 출력된다.
조합을 통해 얻은 최종 플래그는 FLAG{XXXXXXXX}의 형식으로 제출한다.
